===============
Find a Document
===============

.. default-domain:: mongodb

You can find a single document in a collection using the ``FindOne()`` method.

The following example specifies a query filter that matches documents in
the ``movies`` collection with the value "The Room" in the ``title`` field,
and returns the first document that matches.

   Read the :doc:`usage example guide </usage-examples>` to learn how to
